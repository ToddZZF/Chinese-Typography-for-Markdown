## 引用的设计准则

引用的前后距离和段落一样。利用缩进和左边框来区别不同层级的引用。

<style>
    blockquote {
        margin-block-start: 0.5em;
        margin-block-end: 1.5em;
        margin-inline-start: 0;
        margin-inline-end: 0;
        padding-inline-start: 1.7em;
        padding-inline-end: 0;

        border-left: 0.3em solid gray;
    }
</style>
